# -nē

[Alphabetical index](../_Index.md) · [View on Wiktionary](https://en.wiktionary.org/wiki/Reconstruction:Proto-Germanic/-n%C4%93)

## Proto-Germanic

**Reconstruction:** -nē

### Etymology

Possibly from Proto-Indo-European *né-h₁ (“that way; indeed”).
Alternatively, identifiable with the suffix in Sanskrit वि-ना (vi-nā, “without”). The final vowel itself is uncertain; it is usually given as *-ē based on the possibly related forms. See also *fernaz.
Since *a stood before *-nē of most adverbs with this suffix it was rebracketed into *-anē.

### Proto-Indo-European ancestry

**Directly mentioned PIE form(s):** *né-h₁, *néh₁

### Pronunciation

- IPA: `/neː/`

### Adverb

1. _(reconstruction)_ creates ablatival adverbs from X
   - _(ablaut variant of) *ab (“away”) + *-nē → *fanē (“from”)_
   - _þa- (“that”) + *-nē → *þanē (“thence; than”)_
   - _hi- (“this”) + *-nē → *hinē (“hence”)_
   - _aftē (“behind”) + *-nē → *aftanē (“from behind”)_
   - _hi- (“this”) + *jandē (“yonder”) + *-nē → *hindanē (“from beyond”), literally “here from yonder”_
2. _(reconstruction)_ creates emphatic adverbs precisely, just X
   - _swa (“so”) + *-nē → *swanē (> Old Norse svána (“just so”))_
   - _hē₂r (“here”) + *-nē → *hē₂rnē (> Old Norse hérna (“just here”))_

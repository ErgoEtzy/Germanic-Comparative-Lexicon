# *burdą

[Alphabetical index](../_Index.md) · [View on Wiktionary](https://en.wiktionary.org/wiki/Reconstruction:Proto-Germanic/burd%C4%85)

## Proto-Germanic

**Reconstruction:** *burdą

### Etymology

Of uncertain origin; possibly from earlier *brezdą ~ *burzdnaz (“edge, board”) (whence also Old High German borto, Faroese breddi), from Proto-Indo-European *bʰérsdʰ-ō ~ *bʰr̥sdʰ-nés, from *bʰers- (“tip, edge”) + *-dʰeti (compare Sanskrit भृष्टि (bhṛṣtí, “tip, edge”)), as well as Old Irish brot (“goad, spike”). Alternatively perhaps from Proto-Indo-European *bʰerHdʰ-, from *bʰerH- (“to strike, pierce; to work with sharp tools”) + *-dʰeti. Possible cognate with Proto-Slavic *bьrdo (“comb, reed”). Or, compare instead Lithuanian bartà (“board”). Note that all of these explanations require the e-grade byform *bredą to be back-formed, when a root *bred- would otherwise be the simplest explanation.

### Proto-Indo-European ancestry

**Directly mentioned PIE form(s):** *bʰerH-, *bʰerHdʰ-, *bʰerdʰ-, *bʰers-, *bʰérsdʰ-ō ~ *bʰr̥sdʰ-nés, *dʰeti

### Pronunciation

- IPA: `/ˈbur.dɑ̃/`

### Noun

1. _(neuter, reconstruction)_ board, plank
2. _(neuter, reconstruction)_ table
3. _(neuter, reconstruction)_ edge
4. _(neuter, poetic, reconstruction)_ shield

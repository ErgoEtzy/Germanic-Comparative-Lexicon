# *þingą

[Alphabetical index](../../_Index.md) · [View on Wiktionary](https://en.wiktionary.org/wiki/Reconstruction:Proto-Germanic/%C3%BEing%C4%85)

## Proto-Germanic

**Reconstruction:** *þingą

### Etymology

From earlier Proto-Indo-European *tenkóm, likely from *tenk- (“to be suitable”) (compare *þinhaną (“to succeed”) for more cognates), in which case the original meaning was "suitable time; scheduled time", which then shifted to "assembly; issue, matter to discuss".

### Proto-Indo-European ancestry

**Directly mentioned PIE form(s):** *tenkóm

### Pronunciation

- IPA: `/ˈθiŋ.ɡɑ̃/`

### Noun

1. _(neuter, reconstruction)_ appointed time; date; appointment
2. _(neuter, reconstruction)_ meeting; assembly, council
3. _(neuter, reconstruction)_ case, matter, issue (such as what is discussed at an assembly)
4. _(broadly, neuter, reconstruction)_ case, matter, issue (such as what is discussed at an assembly); thing, object
